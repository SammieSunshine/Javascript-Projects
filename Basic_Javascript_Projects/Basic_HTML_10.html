Arrays

  • Collection of data, arranged in rows and columns
  • a group of related things that are stored together in sequence
  • used to organize items in a logical way.
  • Contains "elements"
    • (i.e. in the array [7,3,15] the numbers are "elements"
    • The data within an element is called a "Value
    • Has 2 properties
      • Index
      • Value
  • To identify each element, the easiest method is to start from the left, counting the first element as "0", and then upwards.
  • "Index" is the name for the position of each element
    • 7= Index 0, 3= Index 1, 15= Index 3

Const

  • creates a constant whose scope can be local or global to the block in which it's declared
  • Not immutable, but it's var indentifier can't be reassigned
    • Var contains info expected to change. Const info never changes..

Let

  • Declares variables that can have block scope
  • Similar to var because they both declare variables
  • Variable functions declared with Let inside of curly brackets don't have block scope
    • Can't be accessed outside of the brackets.

Return Statement

  • Stops the execution of a function and returns the value from that function.
